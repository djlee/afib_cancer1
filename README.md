# Scripts used in the manuscript 'Plasma Proteomic Signatures at the Atrial Fibrillation–Cancer Interface'

1_proteomics_cleanup.R - script used to generate filtered olink proteomic data, with imputed data.

2_get_covariates.R - script used to obtain covariates from Data Browser

3_cancer_afib_control.Rmd - A modified analysis of AF + Cancer vs AF AND Cancer + AF vs Cancer, using a more straight forward logistic regression, along with figure-generating code
